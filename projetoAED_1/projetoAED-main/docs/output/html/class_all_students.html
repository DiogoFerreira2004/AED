Note

Notice: p is never higher than 7, because a student cannot be enrolled in more than 7 CUs at the same time.

◆ countYearStudents()

int AllStudents::countYearStudents ( const char &  year) const

Function that counts the number of students in a specific year of the degree.

First finds what year the student is attending (first digit of the class_code is the degree year of that class, and finds the highest of the first digits in the classes the Student is attending).
Then matches that year with the parameter, to count the student if he's in that year.

Parameters
yearYear that students (whose numbers will be counted) are enrolled in.
See also
Student.hpp
Note
Time complexity: O(n*p), where n is the number of Student objects in this->students, and p is the number of Curricular Units the Student has enrolled in.
Notice: p is never higher than 7, because a student cannot be unrolled in more than 7 CUs at the same time.

void AllStudents::printYearStudents ( const char &  year) const

Just like countYearStudents(), first finds what year the student is attending (first digit of the class_code is the degree year of that class, and finds the highest of the first digits in the classes the Student is attending).
