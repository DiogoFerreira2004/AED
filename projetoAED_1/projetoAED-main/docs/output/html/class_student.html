Switches class codes in an existing pair inside the ucClasses vector.

Looks for the pair with the given uc_code parameter and switches the class code for the new class_code parameter.

Parameters
uc_code,class_codeCU code and the new class code.
Note
O(n), where n is the number of CUs a Student is enrolled in.
Notice: p is never higher than 7, because a student cannot be unrolled in more than 7 CUs at the same time.
