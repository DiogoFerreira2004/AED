Comparison function for sorting a vector based on the CU code.

Parameters
turma1,turma2The two Turma objects that are compared while sorting a vector.
Returns
True if:
  • turma1 has a lower CU code than turma2;
  • turmas have the same CU code but turma1 has a lower number of students;
  • turmas have the same CU code and the same number of students but turma1 has a lower class code;
    False otherwise.
See also
Turma.hpp
Note
Time complexity: O(1).
