◆ overlaps()

bool ScheduleSlot::overlaps ( const ScheduleSlot &  slot) const

Check if the ScheduleSlot object that is calling this function overlaps with another existing ScheduleSlot.

If the slots have different weekDays, they doesn't overlap.
If the start of a slot is after the ending of the other, they don't overlap.
If one of the slots' class type is "theoretical class", they don't overlap.

Parameters
slotConst reference to the other ScheduleSlot.
Returns
true if overlaps; false if it doesn't.
Note
Time complexity: O(1).

Calls overlaps() function over every combination of slots between 2 vectors.

Each weekday is mapped to an int in a variable inside the function definition.

Parameters

Returns
true if two slots overlap; false otherwise.
See also
overlaps()
Note
Time complexity: O(n*p), where n is the number of ScheduleSlot objects in a specific class, and p the number of ScheduleSlot objects in another class.
