◆ addScheduleSlot()

void Turma::addScheduleSlot ( const ScheduleSlot &  slot)

Inserts a ScheduleSlot object into the schedule member variable.

If the first element of the schedule vector has default values, it means that the vector only got initialized. The values of that element are changed to the values in the argument and it becomes a valid ScheduleSlot object.
Else, the ScheduleSlot in the argument is inserted into the vector.

Parameters
slotScheduleSlot object to insert into the schedule member variable.
Note
Time complexity: O(1).

friend

Teaches the "less than" operator on how to compare two Turma objects.

A Class object is "less than" another, if their CU code is lower.
If they have the same CU code, a Class object is lower if their class code is lower.

Parameters
turma1,turma2The two Turma objects.
Returns
true if turma1's CU code is lower or the CU code is the same and the class code is lower.
false otherwise.
Note
Time complexity: O(1)
