Detailed Description

Class for Vertices of a graph (Graph class).

Each Vertex object represents an Airport object through its airportCode.

bool Vertex::removeEdgeTo ( Vertex *  dest)
private

Removes an outgoing Edge object from the adj member variable.

Iterates through the adj member variable, looking for the Edge object whose destination (other Vertex object where it's connect to) corresponds to dest parameter; If it finds the Edge, removes it;

Parameters
destPointer to the Vertex object the Edge object connects to.
Returns
true if the Edge object was found;
false if it wasn't;
Note
