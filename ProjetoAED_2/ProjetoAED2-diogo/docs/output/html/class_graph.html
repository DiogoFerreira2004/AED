unordered_set< string > &  artPoints,
int &  i,
const string &  ogVertContent 
) const
private

Auxiliary function that aids the implementation of exercise 3.ix.

Modified Tarjan algorithm to find all the articulation points of the graph.
Articulation points are vertices whose removal increases the number of connected components/isolates parts of graph.

Parameters
vPointer to the current Vertex object being processed during the DFS
artPointsStores all the unique vertex contents (airportCodes) that represent an articulation point.
iUsed to keep track of the order in which the vertices are visited during the DFS traversal and assign a unique number (Vertex object's num member variable) to each Vertex
ogVertContentUsed to identify the "root" of the DFS, to know if it is also an articulation point or not.
Note
Time complexity: O(V + E), where V corresponds to the number of Vertices in the graph and E corresponds to the number of edges in the graph.
See also
articulationPoints()
